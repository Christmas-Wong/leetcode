# 重要问题
- 什么时候反转打印顺序（奇数行，偶数行）
- 奇数行和偶数行的入栈顺序不同
- 如何实现反转打印
## 特殊情况
- 根节点为None
## 边界条件

# 解决方案
- 两个栈
    - 一个栈用来顺序存储
    - 一个栈用来逆序存储
- 打印本行的时候，为下一行做准备
    - 计算下一行打印的个数
    - 为下一行的栈填充数据
