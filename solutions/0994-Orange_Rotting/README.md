# 题目
- 每过一分钟，腐败的橘子上下左右的好橘子就会腐败
- 如果不可以全部腐败，返回-1
- 如果可以全部腐败，返回最快腐败时间

# 方案
- 第一次循环
    - 【队列】记录好橘子的坐标
    - 记录好橘子的数量
- 第二次循环
    - 循环次数即为变质周期
    - 每次弹出一个坏橘子（先进先出），添加新感染的坏橘子，好橘子数量减少
    - 当好橘子数量归0，或者坏橘子队列为空的时候停止循环
- 如果好橘子数量为0，返回时间，否则返回-1

## 边界条件
- 坐标大小限制

## 时间复杂度
- 两次循环遍历<2*O(m*n)
## 空间复杂度
- 最坏空间复杂度S(m*n)
